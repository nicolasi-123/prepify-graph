User Guide

Learn how to search, explore, and analyze Czech business relationships.

How to Search

Path Search (default)

Find the shortest connection between two entities (companies or people).

  1. Type a company name, person name, or ICO in the Start field. Select from the dropdown.
  2. Type and select in the End field.
  3. Click Find Path. The graph and path details will appear below.
You can request up to 5 alternative paths using the "Number of paths" filter.

Multi-Point Routing

Force a path through specific intermediate entities.

  1. Enable "Multi-point routing" checkbox.
  2. Click "+ Add waypoint" to add intermediate stops (up to 5 total).
  3. Select entities for each waypoint, then click Find Multi-Point Path.

Exploration Mode

Browse the graph interactively starting from a single entity.

  1. Enable "Exploration Mode" checkbox.
  2. Search and select one entity, then click Explore Entity.
  3. Right-click any node in the graph to expand its connections.
  4. Use the "Find Path to Node" section to search for a target from visible nodes.

Understanding Path Results

Path Cards

Each path shows a chain of entities connected by relationships. The path length is the number of relationship steps (edges), not entities.

TermMeaning
Path LengthNumber of relationship steps (e.g., A → B → C = 2 steps)

Compare Paths

When multiple paths are found, click "Compare Paths" to see them side-by-side. Entities unique to one path are highlighted with a purple accent.

Filters Explained

FilterWhat it does
Max path lengthLimits the maximum number of steps. Shorter = more direct connections.
Number of pathsHow many alternative routes to return (1, 3, or 5).
Exclude insolventRemoves paths passing through companies with active insolvency proceedings (ISIR).
Exclude foreignRemoves paths passing through non-Czech entities (Cyprus, Netherlands, etc.).
Active onlyOnly shows paths using currently active relationships (ignores historical/deleted ones).

Entity Detail Modal

Click any node to open the detail modal showing:

  • Full name and entity type
  • ID (ICO for companies, generated ID for persons)
  • City and country
  • Insolvency status (if applicable)
  • Mini network graph showing direct connections
  • Connections list grouped by relationship type
Click "Find Paths From Here" to auto-fill the search bar with this entity as the starting point.
